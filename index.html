Utajifunza Nini Kwenye Kozi Hii?

HTML5

Kuunda muundo (structure) wa kurasa za web kwa kutumia semantic HTML.

CSS3

Kupamba na kuipa mtindo kurasa za web, ikiwemo Flexbox na Grid.

Tailwind CSS

Framework ya kisasa ya CSS (utility-first) kwa ajili ya speed na design nzuri.

JavaScript (Basics)

Kuongeza interactivity na kufanya web iwe dynamic (DOM manipulation, events).
